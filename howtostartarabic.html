Coding for Refugee Women

This is an introductory tutorial to show you some of the things you can do with coding.

  1. Right click and click on "Inspect Element". This shows the webpage inspector. On the left you see all elements this webpage consists of. On the right you see the styling of this webpage.
  2. Click on the small arrow in front of < body >...< /body >. This will show all elements of the webpage that belong to the body. If you hover over the lines you see which line corresponds to which area on the webpage.
  3. Click on the class="jumbotron vertical-center". We are going to change the background of this part of the webpage.
  4. On the right side, scroll to the part that starts with .jumbotron { and click on the purple block next to background-color. You should see a colour panel in which you can select a new colour. Go ahead, change the backbround colour to any colour you like! Make sure to scroll to the top of the page to see your changes. Now let's change something else!
  5. Click on the arrow in front of class="container" and select the class="text-center"
  6. On the right side, scroll to the part that starts with .jumbotron h1 { and has a part that says "color:inherit" Change to word inherit to a colour name, for example blue. Scroll up again to see the change you made.
    7. The last change we are going to make is changing the font. For this change we need to add a line of code.
  7. Click next to the line that says "color:inherit". This should create a new line where you can type underneath.
  8. Type the word "font-family", then press ENTER and type the word "times". This changes the font to the font type "times". Just like with the colour, you can change the font to any font you want.
You have now finished the introduction tutorial, well done! If you enjoyed it, and would like to learn more about coding click here.
