Coding for Refugee Women

Coding for Refugee Women is a project that was started at the first Empowerhack London.

At the moment Coding for Refugee Women is mainly a source of information for women who would like to learn how to code.It contains a short and playful introduction to what you can do with coding through the "Inspect Element" feature in the browser. Moreover, it provides an extensive list of resources which contains links to existing coding schools and groups. A lot of courses offered on these pages can be followed online.

In the future, we are planning to extend this website to a platform on which you can find mentors who can help you to learn how to code. To receive updates about new features and other news sign up for the newsletter here: http://eepurl.com/bKk5cf.
